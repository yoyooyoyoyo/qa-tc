- 대상은 착한의사 App 병원/검진 상세, 예약/결제/예약내역, 쿠폰 사용, 예약 접수 실패, 어드민 쿠폰관리, 회원상세 보유쿠폰 영역이다.

## App 병원/검진 상세

- 병원/검진 상세에는 상단 이미지, 병원정보, 병원공지, 쿠폰 박스, 스크롤 포인트 탭바가 노출된다.
- 주요 섹션은 `검사정보`, `예상소요시간`, `내시경실운영현황`, `당일 검사결과 설명`, `검진비교`, `병원소개`, `의료진`, `병원위치`, `연령별통계`, `검진피드`, `방문자 평점`, `같은병원 다른상품`, 하단 주의사항이다.
- `쿠폰다운전` 상태가 별도 케이스로 확인된다.
- `쿠폰드롭` 화면에는 회사명 환영 문구, B2B 이용자 선택 탭, 이용자별 사용 횟수 예시(`사용 1/2`), 이용자 관리 진입, 찜한검진 B2B 배너가 포함된다.
- 내시경실 운영현황에는 도움말 아이콘과 툴팁이 있으며, 툴팁 노출/닫기 여부를 별도 확인한다.
- 하단 CTA는 `fullwidth/h64_new` 또는 `fullwidth/h64` 형태로 확인된다.

- `UI-INFO-BOOK-1001_비멤버십_선결제`는 비멤버십 선결제 예약/결제 대표 케이스이다.
- 예약 화면 구성은 예약카드+날짜, 방문자정보, 방문자특이사항, 멤버십/할인혜택, 결제정보, 결제방법, 하단 버튼으로 나뉜다.
- 방문자정보 영역에는 `검진 예약내역` 제목과 알림 배너(`bar/alarm/noti_mdch04`)가 있다.
- 방문자특이사항 영역에는 수정사항 입력/확인 영역이 있으며, 문구 예시는 `자세한 수정사항을 입력해주세요!`이다.
- 결제정보에는 펼침/접힘 가능한 결제 상세(`inter/sm24/payment/drop`, `box_payment_drop`)가 포함된다.
- 결제 상세 행에는 도움말 아이콘, 금액, 펼침 화살표가 조합된다.
- 결제방법 영역에는 카드결제 노출 케이스가 확인된다.
- 예약확정 선결제 화면은 4단계 진행 상태(progress), 예약카드+날짜, 방문자정보, 방문자특이사항, 할인혜택, 결제정보, 결제방법, 안심케어, 하단 버튼/주의사항으로 구성된다.
- 예약확정 선결제 App 화면에는 `안심케어` 영역이 있으며 `100% 무료` 배지, 용종 발견 보장 문구, 상세 진입 버튼이 확인된다.
- 예약확정 화면 하단 버튼은 2개 버튼 병렬 + 1개 전체폭 버튼 조합으로 확인된다.
- 하단 주의사항 문구는 `영수증에는 검진일이 표시되지 않아요. 검진일은 예약내역의 [예약 날짜]를 확인해주세요.`와 `병원에서 결제한 영수증은 병원에 문의해주세요.`이다.

## App 쿠폰 사용

- 예약/결제 화면에는 `쿠폰사용`, `사용가능쿠폰없음`, `쿠폰있으나 사용하지 않음`, `쿠폰사용툴팁` 케이스가 분리되어 있다.
- 쿠폰 사용 영역은 할인혜택/멤버십사용 영역 안에서 쿠폰명 또는 혜택 태그, 도움말 아이콘, 쿠폰 적용 결과를 노출한다.
- 쿠폰 사용 툴팁은 별도 화면 상태로 제공되며, 도움말 아이콘 탭 또는 hover/tap으로 노출되는 것으로 보인다.
- 사용 가능한 쿠폰이 없을 때에도 결제금액/결제 상세 영역은 유지되며, 쿠폰 영역의 상태만 달라지는지 확인한다.
- 쿠폰이 있으나 사용하지 않은 케이스는 할인금액/최종 결제금액 반영 여부를 쿠폰사용 케이스와 비교한다.

- 실패 화면의 하단 CTA는 `fullwidth/h64`로 확인되며, 버튼 문구/동작은 TC 작성 직전에 확인한다.

## Admin 쿠폰 상태 플로우

- 쿠폰 상태 축은 `대기`, `발행시작`, `운영중`, `중지`, `종료`이다.
- 쿠폰발행 등록 후 미래 날짜 설정(날짜지정/기간설정) 시 `대기` 상태가 된다.
- 즉시 발행 시 `발행시작`을 거쳐 `운영중`이 된다.
- 지정 날짜에 도달하면 대기 상태에서 발행/운영 상태로 전환된다.
- `대기`는 발행 전 상태이며 쿠폰 정보 수정 및 발행 취소가 가능한 상태로 설명되어 있다.
- 관리자가 직접 설정하면 `중지` 상태로 전환될 수 있다.
- 발행기간 종료, 쿠폰 사용기간 종료, 설정한 발행수량 완전 소진 시 `종료` 상태가 된다.
- 발급기간 내 + 쿠폰 사용기간 내는 `운영중`으로 표기한다.
- 발급기간 종료 + 쿠폰 사용기간 내도 `운영중`으로 표기하는 정책 메모가 있다.
- 발급기간 종료 + 쿠폰 사용기간 종료는 `종료`로 표기한다.

## Admin 쿠폰 목록

- 쿠폰관리 목록에는 기간 필터, 검색창, 페이지당 보기 수 선택, 신규쿠폰등록, 일괄 중지 버튼이 있다.
- 기간 필터 기준 예시는 `발행일 기준`이며, 기간 선택 UI가 제공된다.
- 검색 placeholder는 `쿠폰명, 최근 작성자로 검색하기`이다.
- 탭/필터 상태는 `전체`, `선택`, `대기`, `운영중`, `종료`, `중지`가 확인된다.
- 목록 컬럼은 쿠폰명, 쿠폰금액(원), 발급기간, 등록일시, 수정일시, 최근 작성자, 상태로 구성된다.
- 쿠폰금액 예시는 `50,000`, `100,000`, `150,000`, `10,000`이다.
- 선택 체크박스가 있는 목록이며, 선택 수와 일괄 중지 액션을 함께 확인한다.
- 상태 변경 시 `confirm/상태변경/체크박스o` 확인 팝업이 노출되는 케이스가 있다.

## Admin 쿠폰 등록/상세/수정

- 쿠폰등록 화면 진입 경로는 `어드민/서비스설정_쿠폰관리 > 쿠폰등록`이다.
- 상단 액션은 `입력초기화`, `등록완료`이다.
- 쿠폰상세 화면 진입 경로는 `어드민/서비스설정_쿠폰관리 > 쿠폰상세`이다.
- 쿠폰상세 상단 액션은 `쿠폰중지`, `수정하기`이며, 최근 수정/작성 정보 예시는 `2025-09-30 13:12 장정은`이다.
- 쿠폰수정 화면 진입 경로는 `어드민/서비스설정_쿠폰관리 > 쿠폰수정`이다.
- 쿠폰수정 상단 액션은 `수정취소`, `수정완료`이다.
- 공통 기본정보 항목은 `쿠폰명`, `쿠폰안내`, `쿠폰상태`, `관리자메모`이다.
- 쿠폰명은 `15자 이내로 쿠폰명을 입력해주세요.` placeholder가 확인된다.
- 쿠폰안내 placeholder는 `쿠폰 안내에 들어갈 텍스트를 입력해주세요.`이다.
- 쿠폰안내 예시는 `사용기간 내 사용 가능`, `건강검진 상품 결제 시 1회 사용 가능`, `다른 쿠폰/할인과 중복 적용 불가`, `쿠폰 금액 초과분은 환불되지 않음`, `사용기간 만료 시 자동 소멸`이다.
- 발급조건 항목은 회원유형, 검진타입, 발급유형, 총 수량, 1인당 수량, 발급기간이다.
- 회원유형 옵션은 `B2C`, `B2B`, `B2B2C`가 확인된다.
- 검진타입 옵션은 `종합검진`, `채용검진`이 확인된다.
- 발급유형은 `다운로드쿠폰`으로 확인된다.
- 총 수량 placeholder는 `수량 입력(ex. 100,000)`이며, `제한없음` 옵션이 있다.
- 1인당 수량 placeholder는 `예: 1 이상 숫자 입력`이며, `수량제한안함` 옵션이 있다.
- 사용조건 항목은 회원조건, 적용항목, 할인금액, 사용기간이다.
- 회원조건은 `전체회원`, 적용항목은 `착한의사 서비스이용료`로 확인된다.
- 할인금액은 `1,000원 이상 입력` placeholder와 예시 `50,000원`이 확인된다.
- 발급기간/사용기간은 날짜 범위 선택이며 예시는 `2025-09-30 ~ 2025-10-21`이다.
- 등록/상세/수정 화면 모두 상태 변경 또는 저장 관련 확인 팝업이 있으므로, 저장 전/후와 취소 동작을 분리한다.

- 회원상세에는 회원 기본정보, 이용자 정보, 멤버십 이력, 검진이력, 보유쿠폰, 보유마일리지, 펫 검진 탭이 있다.

- 쿠폰사용일은 미사용 또는 미기록 시 `-`로 표기되는 케이스가 있다.
- 멤버십 이력에는 가입일자, 잔여횟수, 상태가 있으며 상태 예시는 `사용중`, `만료`이다.
