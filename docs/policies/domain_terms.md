# Domain Terms

## 화면/메뉴 경로

- 기업검진 > 종합검진 목록: 기업 고객 대상 종합검진 상품 또는 예약 목록 화면
- 기업검진 > 종합검진 상세: 종합검진 항목 상세 확인 화면
- 어드민 > 기업검진 관리: 기업검진 상품/예약/상태를 관리하는 어드민 화면

## 사용자/권한

- 사용자: 서비스를 이용하는 일반 사용자
- 관리자: 어드민에서 데이터를 관리하는 운영자
- 기업 담당자: 기업검진 대상자 또는 예약 현황을 관리하는 담당자

## 상태값 예시

- 예약가능: 사용자가 예약을 진행할 수 있는 상태
- 예약불가: 예약 조건 미충족 또는 마감으로 예약할 수 없는 상태
- 대기: 요청은 생성되었으나 확정되지 않은 상태
- 완료: 요청 또는 처리가 정상 완료된 상태
- 취소: 사용자 또는 운영자에 의해 취소된 상태
- 실패: 시스템 오류 또는 validation 실패로 처리가 완료되지 않은 상태

## 쿠폰/가격

- 다운로드쿠폰: 사용자가 상품 상세 또는 예약 과정에서 직접 다운로드해 보유할 수 있는 쿠폰
- 코드쿠폰: 쿠폰 코드 입력 또는 외부 지급 등으로 이미 보유 상태가 되는 쿠폰
- 보유중: 사용자가 쿠폰을 다운로드했거나 지급받아 사용할 수 있는 상태
- 쿠폰적용가: 쿠폰 할인 적용 후 사용자가 결제할 금액
- 정가: 쿠폰 또는 할인 적용 전 기준 금액

## 용어 작성 규칙

- 기획서 용어와 회사 표준 용어가 다르면 회사 표준 용어를 우선 사용한다.
- 신규 용어가 나오면 이 문서에 정의를 추가한다.
- 같은 의미의 용어가 여러 개면 테스트케이스에서는 하나의 표준 용어로 통일한다.
