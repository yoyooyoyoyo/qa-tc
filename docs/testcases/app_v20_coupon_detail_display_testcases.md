# App v20 검진상세 쿠폰 노출 개선 TC

Source:

- Figma `착한의사APP`, node `87:11517`
- 기획서 섹션: `[쿠폰] 검진상세 노출 개선`

## 범위

- App 검진상세 화면의 쿠폰 카드 노출 개선을 검증한다.
- 주요 상태는 `다운로드 전`, `보유중`, `코드쿠폰 보유중`이다.
- AS-IS/TO-BE 비교 기준으로 기존 다운로드 후 쿠폰 카드가 흐리게 보이던 표현이 TO-BE 보유중 상태로 변경되는지 확인한다.
- 쿠폰 금액 예시는 Figma 기준 `정가 165,000원`, `쿠폰적용가 150,000원`이다.

## 테스트케이스

### TC-APP-COUPON-001

- title: 검진상세 진입 시 다운로드 가능한 쿠폰이 있으면 병원정보 영역에 쿠폰 카드가 노출되는지 확인
- 테스트 진행 화면: App > 검진상세
- precondition:
  - 로그인한 사용자
  - 검진상세 상품에 다운로드 가능한 쿠폰이 존재함
  - 사용자가 해당 쿠폰을 아직 보유하지 않음
- steps:
  1. App에서 쿠폰 대상 검진상품 상세 화면에 진입한다.
  2. 상단 병원정보와 병원공지 영역 하단을 확인한다.
- expected result:
  - 병원정보와 병원공지 하단에 쿠폰 카드가 노출된다.
  - 쿠폰 카드 하단에 검사정보 스크롤 탭바가 이어서 노출된다.
  - 쿠폰 카드와 스크롤 탭바가 겹치거나 가려지지 않는다.
- priority: High
- test_type: Functional
- notes: Figma TO-BE `다운로드 전` 상태 기준
